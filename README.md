# BrightStar Portal — พอร์ทัลผู้ปกครอง / โรงเรียน / เจ้าหน้าที่ / ผู้ดูแลระบบ

หน้าเข้าสู่ระบบกลางของ **BrightStar Assessment Institute (BSI)** เลือกบทบาทเพื่อเข้าสู่หน้าของแต่ละกลุ่ม ดูผล Star Path, เกียรติบัตร, ข้อมูลการจัดสอบ และคำอธิบายหลักการ/แนวทางการสอบ (พร้อมอินโฟกราฟิก)

## โครงสร้าง
- `index.html` — **หน้าเข้าสู่ระบบอย่างเดียว** เลือกบทบาท: ผู้ปกครอง/นักเรียน · โรงเรียน · เจ้าหน้าที่/ครู · ผู้ดูแลระบบ → routing ไปหน้าของแต่ละกลุ่ม
- `parent.html` — ผู้ปกครอง/นักเรียน: คะแนน 5 ด้าน + เรดาร์ + ระดับดาว + เกียรติบัตร (จากลิงก์ผลเฉพาะคน)
- `school.html` — โรงเรียน: ภาพรวมทั้งโรงเรียน (KPI, ค่าเฉลี่ยรายด้าน, เปรียบเทียบรายชั้น)
- `staff.html` — เจ้าหน้าที่/ครู: เครื่องมือจัดสอบ/กรอกผล (เชื่อมเว็บหลัก + AppSheet)
- `admin.html` — ผู้ดูแลระบบ/ผู้บริหาร: ภาพรวมทั้งระบบ + เครื่องมือจัดการ
- `examinfo.js` — ส่วน "หลักการและแนวทางการสอบ" + อินโฟกราฟิก + Star Path 7 ระดับ + ข้อมูลการจัดสอบ (ใช้ร่วมทุกหน้า **ยกเว้นหน้า login**)
- `logo.png` — โลโก้สถาบัน

## การเข้าสู่ระบบ
- **ผู้ปกครอง/นักเรียน** — วางลิงก์/รหัสผลที่สถาบันส่งให้ (เว้นว่างได้เพื่ออ่านข้อมูลทั่วไป)
- **โรงเรียน / เจ้าหน้าที่ / ผู้ดูแลระบบ** — กรอกรหัสเข้าพอร์ทัล (แก้ไขได้ในไฟล์ `index.html` ที่ตัวแปร `ACCESS`)
  - ค่าเริ่มต้น: โรงเรียน `school2569` · เจ้าหน้าที่ `staff2569` · ผู้ดูแลระบบ `admin2569`

> รหัสพอร์ทัลเป็น "รหัสอำนวยความสะดวก" (client-side) ไม่ใช่ระบบรักษาความปลอดภัยจริง — หน้าเหล่านี้ไม่มีข้อมูลส่วนตัว มีแต่ลิงก์/ตัวอย่าง  
> **ข้อมูลส่วนตัวจริง** เข้าถึงได้หลังล็อกอิน **ระบบกลาง (AppSheet)** ซึ่งแสดงเฉพาะข้อมูลในสิทธิ์ของผู้ใช้ (Security Filter ตามอีเมล)
