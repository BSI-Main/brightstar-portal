# BrightStar Portal — พอร์ทัลผู้ปกครอง & โรงเรียน

หน้าเว็บสำหรับ **ผู้ปกครอง** และ **โรงเรียน** เข้าดูผลประเมิน Star Path, เกียรติบัตร, ข้อมูลการจัดสอบ และคำอธิบายข้อสอบ 5 ห้อง ของ **BrightStar Assessment Institute (BSI)**

## โครงสร้าง
- `index.html` — หน้าแรกพอร์ทัล: เลือกบทบาท (ผู้ปกครอง/โรงเรียน) + ปุ่มล็อกอินจริงไป AppSheet + ช่อง "ดูผลด่วน" ด้วยรหัส + ตารางสอบ + คำอธิบายข้อสอบ
- `parent.html` — แดชบอร์ดผู้ปกครอง: ถอดรหัสลิงก์ผล (`#d=`) → คะแนน 5 ด้าน, กราฟเรดาร์, ระดับดาว, เกียรติบัตร
- `school.html` — แดชบอร์ดโรงเรียน: ปุ่มล็อกอินจริง + ตัวอย่างภาพรวม (KPI, ค่าเฉลี่ยรายด้าน, เปรียบเทียบรายชั้น)
- `logo.png` — โลโก้สถาบัน

## การล็อกอิน (สถาปัตยกรรมแบบผสม)
เว็บนี้เป็น static (GitHub Pages) จึงไม่เก็บข้อมูลส่วนตัวและไม่ทำ auth จริงบนหน้าเว็บ
- **ข้อมูลสด** → ปุ่ม "เข้าสู่ระบบ" พาไปล็อกอินที่ **ระบบกลาง AppSheet** ซึ่งแสดงเฉพาะข้อมูลในสิทธิ์ของผู้ใช้ (Security Filter ตามอีเมล)
- **ดูผลด่วน** → ผู้ปกครองวางลิงก์/รหัสเฉพาะที่สถาบันส่งให้ ระบบถอดรหัสแสดงผลของบุตรหลานเท่านั้น (ไม่มีฐานข้อมูลรวมบนเว็บ)
